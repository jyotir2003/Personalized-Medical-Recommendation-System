Machine Learning Technology

Our system employs a Decision Tree Classifier, a supervised learning algorithm that creates a model predicting the value of a target variable by learning simple decision rules inferred from the data features.

The model was trained on a comprehensive dataset containing symptoms and corresponding diseases. Through this training process, the algorithm learned to recognize patterns and relationships between various symptoms and potential medical conditions.

When you input your symptoms, the model evaluates them against what it has learned to determine the most likely diagnoses. This approach balances accuracy with interpretability, allowing us to understand how conclusions are reached.

Flask Backend Technology

Flask is a micro web framework written in Python. It's lightweight yet powerful, making it ideal for our application's needs.

The backend handles several critical functions:

  • Receiving symptom data from the frontend
  • Loading and utilizing the trained machine learning model
  • Processing the data through the model to generate predictions
  • Returning results to the frontend for display

Using Flask allows us to efficiently connect our Python-based machine learning model with the web interface, creating a seamless experience for users.

Frontend Development Technology

The frontend is built using modern web development technologies:

  • HTML5: Provides the structure and content of our web pages
  • CSS3: Controls the visual presentation and layout
  • JavaScript: Enables interactive elements and dynamic behavior
  • Bootstrap: A responsive framework that ensures our application looks great on all devices
  • Font Awesome: Provides icons for visual enhancement

These technologies work together to create an intuitive interface where users can easily input their symptoms and receive clear, actionable information.
